# 🛡️ Ransomware Detection Using Machine Learning & Deep Learning

## 📌 Project Overview
This project detects **Ransomware attacks** using Machine Learning and Deep Learning algorithms.  
It classifies system behavior into:
- **Benign (0)**
- **Ransomware (1)**

---

## 🚀 Features
- Data preprocessing and normalization
- Visualization of dataset
- Multiple ML & DL models:
  - SVM
  - KNN
  - Decision Tree
  - Random Forest
  - XGBoost
  - DNN
  - LSTM
  - CNN
- Performance evaluation:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix
- Graph comparison of all algorithms
- Prediction on test data

---

## 🗂️ Dataset

## 📊 Workflow
1. Load dataset  
2. Handle missing values  
3. Normalize features  
4. Shuffle dataset  
5. Train-test split (80:20)  
6. Train models  
7. Evaluate performance  
8. Compare results  
9. Predict new data
